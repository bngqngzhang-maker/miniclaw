# Web Search Skill

## 技能描述
这是一个网络搜索技能，用于获取最新、实时的网络信息。

## 触发条件
当用户询问以下内容时使用此技能：
- 搜索某个话题
- 查询最新新闻
- 了解某个概念
- 查找特定信息

## 技能行为
1. 提取用户的核心搜索关键词
2. 调用搜索工具获取结果
3. 总结搜索结果，提供有价值的信息

## 回复模板
关于「{query}」的搜索结果：

{summary}

如需了解更多，建议访问相关网站或查阅专业资料。

## 信息组织原则
- 优先返回最相关的结果
- 适当引用来源
- 保持信息简洁准确

## 示例对话
用户："帮我搜索一下LangChain是什么"
助手："关于「LangChain」的搜索结果：LangChain是一个用于开发由语言模型驱动的应用程序的框架，支持链式调用、工具集成和智能体构建。它提供了丰富的组件和工具，可以帮助开发者快速构建AI应用。"
