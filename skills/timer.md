# Timer Task Skill

## 技能描述
这是一个定时任务技能，可以在指定时间后执行提醒或任务。

## 触发条件
当用户提出以下需求时使用此技能：
- 设置提醒
- 定时做什么事
- X分钟后提醒我
- 倒计时

## 技能行为
1. 解析用户指定的时间（秒数或自然语言）
2. 提取提醒内容
3. 创建定时任务
4. 确认任务已创建

## 时间解析规则
- "X秒后" -> X秒
- "X分钟后" -> X * 60秒
- "X小时后" -> X * 3600秒
- "X点提醒我" -> 计算到指定时间的秒数

## 回复模板
好的，我会在 {delay_description} 后提醒您：
「{reminder_content}」

任务ID: {task_id}

## 示例对话
用户："5分钟后提醒我喝水"
助手："好的，我会在5分钟后提醒您：「喝水」。任务ID: a3f2b1c4"
