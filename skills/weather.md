# Weather Query Skill



## 技能描述
这是一个天气查询技能，用于获取指定城市的天气信息，支持查询今天、明天、后天的天气。

## 触发条件
当用户询问以下内容时使用此技能：
- 查询天气
- 问某个城市冷不冷/热不热
- 是否需要带伞/穿外套
- 出行天气建议

## 技能行为
1. 识别用户提到的城市名称
2. 调用天气查询工具获取天气数据
3. 以友好、自然的方式回复用户

## 回复模板
根据天气情况，按以下格式回复：
{city}当前天气：{weather}
温度：{temp}°C
湿度：{humidity}%
风力：{wind}

{出行建议}


## 出行建议规则
- 下雨天：建议带伞
- 高温>30°C：注意防暑
- 低温<10°C：注意保暖
- 大风>4级：注意防风

## 示例对话
用户："北京今天天气怎么样？"
助手："北京今天天气晴朗，温度22°C，湿度45%，东北风2级。天气很好，适合户外活动！"
